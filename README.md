Это проект для курса Java AQA

проект содержит терию и практические задания

как делать практические задания:

1. Клонируем себе гит проект - git clone https://github.com/DzmitryBInnowise/javaCourse.git
2. Идем в папку practice
3. Находим соответствующее задание
4. Идем в папку practiceAnswers
5. Создаем java файл с номером лекции и своим именем e.g. - FirstSessionVasyaPupkin.java
6. Решаем задания и делаем комит: 
git checkout -b "<branch Name>" -> 
git add "<fileName>" -> 
git commit -m "Your Message here" -> 
git push origin "<branch Name>"
7. на юае сделать пул реквест
